Use Scalekit authentication with Model Context Protocol (MCP) servers for AI applications.

## What is MCP

MCP (Model Context Protocol) enables AI models to securely access external data sources and tools.

## Scalekit MCP integration

Scalekit provides authentication and authorization for MCP servers:
- User authentication
- OAuth token management
- Permission scoping
- Multi-provider support
