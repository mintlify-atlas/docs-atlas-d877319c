## Key features

### Multi-product authentication platform

Scalekit offers flexible authentication solutions that work independently or together:

- **Full-stack authentication** - Complete auth solution with hosted login, sessions, and user management
- **Enterprise SSO** - SAML and OIDC integrations for enterprise customers
- **SCIM provisioning** - Automated user and group synchronization
- **Agent authentication** - OAuth flows for AI agents and LLM applications
- **MCP authentication** - Secure Model Context Protocol server connections

### Enterprise-ready features

- **Self-service admin portal** - Let enterprise customers configure SSO and SCIM themselves
- **Custom domains** - Use your own domain for authentication flows
- **Webhooks** - Real-time notifications for user events, SSO connections, and directory changes
- **RBAC** - Role-based access control with custom roles and permissions
- **Session policies** - Configurable session duration, timeout, and security settings

## Use cases

### B2B SaaS applications

Add enterprise authentication features required by large customers:
- Enterprise SSO for secure single sign-on
- SCIM provisioning for automated user management
- Custom domains for white-label experiences
- Admin portals for customer self-service

### AI and agent platforms

Secure AI applications with specialized authentication:
- OAuth flows for AI agents accessing business tools
- MCP server authentication for Claude Desktop and other clients
- Tool authorization with scope-based permissions
- Connected account management for multi-provider access

### Multi-tenant applications

Manage authentication across organizations:
- Organization-based user isolation
- Per-organization SSO configurations
- Group-based role assignment
- Cross-organization user identity management
