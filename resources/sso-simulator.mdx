The SSO Simulator helps you test SSO flows during development.

## Features

- Test SAML 2.0 flows
- Test OIDC flows
- Customize user attributes
- Test error scenarios
- No IdP configuration required

## Using the simulator

1. Create test organization with domain `@example.com`
2. Generate authorization URL with test email
3. User redirected to simulator login page
4. Enter any credentials (all accepted)
5. Complete authentication flow

## Custom attributes

Test custom SAML/OIDC attributes:

- Modify returned user claims
- Test attribute mapping
- Verify profile synchronization
