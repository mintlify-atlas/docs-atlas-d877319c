Scalekit supports SCIM 2.0 provisioning with these directory providers.

## Supported providers

### Okta

- User provisioning
- Group sync
- Attribute mapping
- Real-time updates

### Microsoft Entra ID

- User provisioning
- Group sync
- Attribute mapping
- Delta sync

### Google Workspace

- User provisioning
- Group sync
- Attribute mapping

### JumpCloud

- User provisioning
- Group sync
- Attribute mapping

### OneLogin

- User provisioning
- Group sync
- Attribute mapping

## SCIM operations

All providers support:

- Create users
- Update users
- Delete users
- Create groups
- Update group membership
- Delete groups
