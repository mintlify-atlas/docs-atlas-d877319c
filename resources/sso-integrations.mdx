Scalekit supports SSO integration with these identity providers.

## Enterprise providers

### Okta

- SAML 2.0 support
- OIDC support
- SCIM provisioning
- MFA integration

### Microsoft Entra ID (Azure AD)

- SAML 2.0 support
- OIDC support
- SCIM provisioning
- Conditional access

### Google Workspace

- SAML 2.0 support
- OIDC support
- SCIM provisioning

### JumpCloud

- SAML 2.0 support
- OIDC support
- SCIM provisioning

### OneLogin

- SAML 2.0 support
- OIDC support
- SCIM provisioning

## Generic providers

### SAML 2.0

Any SAML 2.0 compliant identity provider.

### OIDC

Any OpenID Connect compliant identity provider.
