Track updates, new features, and improvements to Scalekit.

## 2024

### March 2024

#### New features

- **Agent Auth**: OAuth authentication for AI agents
- **MCP support**: Model Context Protocol integration
- **Enhanced session management**: Remote session revocation

#### Improvements

- Faster token refresh performance
- Improved webhook reliability
- Enhanced IdP Simulator

#### Bug fixes

- Fixed SAML certificate validation
- Resolved token refresh race condition
- Fixed SCIM group sync edge cases

### February 2024

#### New features

- **SCIM 2.0**: Automatic user provisioning
- **Group-based roles**: Map IdP groups to app roles
- **Webhook events**: Real-time event notifications

#### Improvements

- Dashboard UI enhancements
- SDK performance improvements
- Documentation updates

### January 2024

#### New features

- **Multi-device sessions**: Manage sessions across devices
- **Session policies**: Configurable timeouts
- **Token encryption**: Built-in token encryption helpers

#### Improvements

- Enhanced security features
- Better error messages
- Faster API responses

## 2023

### December 2023

#### Initial release

- Full-stack authentication
- Enterprise SSO (SAML & OIDC)
- Social authentication
- Session management
- Organization management
- Multi-tenant support

## Upcoming features

- Advanced MFA options
- Passwordless authentication improvements
- Enhanced analytics dashboard
- Custom domains
- Mobile SDK improvements
