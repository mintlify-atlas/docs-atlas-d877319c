Scalekit supports authentication with these social providers.

## Available providers

### Google

- OAuth 2.0 support
- Profile information
- Email verification

### Microsoft

- OAuth 2.0 support
- Azure AD integration
- Office 365 accounts

### GitHub

- OAuth 2.0 support
- Developer-friendly
- Organization membership

### LinkedIn

- OAuth 2.0 support
- Professional profiles
- Company information

### Facebook

- OAuth 2.0 support
- Social graphs
- User demographics

### Apple

- Sign in with Apple
- Privacy-focused
- Email relay

## Configuration

Each provider can use:

- **Scalekit credentials**: Quick start, no configuration
- **Your credentials**: Production use, custom branding
