## PATCH /api/v1/organizations/{id}

Updates an organization's display name, external ID, or metadata. Requires a valid organization identifier. Region code cannot be modified through this endpoint.

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier of the organization to be updated.
  
  ```
  org_59615193906282635
  ```
</ParamField>

### Body Parameters

<ParamField body="display_name" type="string">
  Name of the organization to display in the UI. Must be between 1 and 200 characters.
  
  ```
  Acme Corporation
  ```
</ParamField>

<ParamField body="external_id" type="string">
  Your application's unique identifier for this organization, used to link Scalekit with your system.
  
  ```
  tenant_12345
  ```
</ParamField>

<ParamField body="metadata" type="object">
  Custom key-value pairs to store with the organization. Keys must be 3-25 characters, values must be 1-256 characters. Maximum 10 pairs allowed.
  
  ```json
  {
    "industry": "technology",
    "plan": "enterprise"
  }
  ```
</ParamField>

## Response

<ResponseField name="organization" type="object">
  The updated organization with all current details reflected in the response.

  <Expandable title="organization object">
    <ResponseField name="id" type="string">
      Unique scalekit-generated identifier
    </ResponseField>

    <ResponseField name="display_name" type="string">
      Updated name of the organization
    </ResponseField>

    <ResponseField name="external_id" type="string">
      Updated external identifier
    </ResponseField>

    <ResponseField name="metadata" type="object">
      Updated metadata key-value pairs
    </ResponseField>

    <ResponseField name="region_code" type="string">
      Geographic region code (immutable)
    </ResponseField>

    <ResponseField name="create_time" type="string">
      Timestamp when the organization was created (ISO 8601)
    </ResponseField>

    <ResponseField name="update_time" type="string">
      Timestamp when the organization was last updated (ISO 8601)
    </ResponseField>

    <ResponseField name="settings" type="object">
      Organization settings
    </ResponseField>
  </Expandable>
</ResponseField>
